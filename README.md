# 后台服务器

## 简介

该项目是整个项目的后台服务器部分，使用nodejs搭建服务器。

项目不包含任何界面，访问方式是:

```
http://ip:端口/数据库的集合名/操作方法
```

具体接口说明参看项目中的文档。

## 数据库连接配置

修改package.json文件中db项。其中url是数据库连接地址，dbname是数据库名。

## 下载依赖包

进入项目的根目录，键入：

```
npm install 
```
## 启动服务器

方法一：

进入项目的根目录，键入：


```
npm start
```

方法二：

在任意目录键入：

```
node /项目目录/bin/www
```
